Métodos de Manipulación de DOM en JavaScript

Método Descripción Ejemplo
createElement() Crea un nuevo elemento HTML (pero no lo añade al DOM). const div = document.createElement('div');
appendChild() Añade un nodo como el último hijo de un elemento. parent.appendChild(child);
prepend() Inserta un nodo al principio de un elemento padre. parent.prepend(child);
insertBefore() Inserta un nodo antes de un nodo específico hijo de un elemento. parent.insertBefore(newNode, referenceNode);
insertAdjacentHTML() Inserta HTML en una posición específica (beforebegin, afterbegin, beforeend, afterend). element.insertAdjacentHTML('beforeend', '<p>Texto</p>');
innerHTML Reemplaza el contenido HTML de un elemento. element.innerHTML = '<p>Texto nuevo</p>';
removeChild() Elimina un nodo hijo de un elemento. parent.removeChild(child);
remove() Elimina el elemento del DOM. element.remove();
replaceChild() Reemplaza un nodo hijo por otro nodo en el mismo elemento padre. parent.replaceChild(newChild, oldChild);
cloneNode() Crea una copia de un nodo. Puede copiar con o sin sus hijos (true para copiar hijos). const clone = element.cloneNode(true);
setAttribute() Establece o modifica el valor de un atributo en un elemento. element.setAttribute('class', 'nuevo');
removeAttribute() Elimina un atributo específico de un elemento. element.removeAttribute('class');
classList.add() Añade una o más clases a un elemento. element.classList.add('nuevaClase');
classList.remove() Elimina una o más clases de un elemento. element.classList.remove('clase');
classList.toggle() Añade o elimina una clase, dependiendo de si ya está presente. element.classList.toggle('clase');
textContent Establece o devuelve el contenido de texto de un nodo (sin HTML). element.textContent = 'Texto nuevo';
